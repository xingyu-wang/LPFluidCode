void BoundaryFactory::registerBoundary ( std::string  name,
GeoCreateFunc  func 
)

Registers (links) the boundary name name with the function func for creating objects in the Boundary family.

After registration, name can be used as an argument in the createBoundary member function for creating objects of the linked type in the Boundary family

Parameters
[in]namethe boundary name
[in]functhe function pointer pointing to the function that creates objects of a specific type in the Boundary family
Returns
None
Note
Instead of using this function directly, consider using the BoundaryRegistrar class for the purpose of linking a boundary name and a specific class in the Boundary family. The function is kept public in case one wants to use it directly
